# Let's Learn About Satellites — Course Assets

This repository stores image and media assets for the "Let's Learn About Satellites" interactive e-learning course.

## Structure

```
/logo                 ← Space faculty logo
/assets
  /ADCS               ← All images for the ADCS chapter
```

## Usage

Images are referenced directly in HTML course components via their raw GitHub URL.
Please do not rename files after they have been linked to a component.

## Notes

- New chapter folders will be added to `/assets` as the course grows.
- Maintained by the Space Faculty course production team.
